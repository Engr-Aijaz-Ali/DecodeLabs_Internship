# ATM Interface using Java GUI

A simple ATM Interface application developed using Java Swing. This project simulates basic banking operations through a graphical user interface.

## Features

* Balance Inquiry
* Cash Deposit
* Cash Withdrawal
* Transaction Validation
* User-Friendly GUI

## Technologies Used

* Java
* Swing (GUI)
* Object-Oriented Programming (OOP)

## Purpose

This project was developed to practice Java GUI development and implement core banking functionalities in a desktop application.
